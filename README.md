# Task Management Backend

This is the backend for the Task Management application, built with **Node.js**, **Express**, and **MongoDB**, **React**, **Zustand** 
It provides authentication, board management, role management, and board list APIs.

---

## Features

- User Signup, Login, and Logout
- JWT-based authentication
- Check user authentication status
- CRUD for Boards
- CRUD for Board Lists
- Role management for admins
- CORS configured for local development and deployed frontend

---

## Tech Stack

- Node.js
- Express.js
- MongoDB / Mongoose
- bcrypt / JWT
- Ract
- bootstrap
- zustand (state management)
